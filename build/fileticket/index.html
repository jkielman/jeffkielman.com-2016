Roles

Daniel Jenner From OTT Legal, hired me to design and build Fileticket. A web application used to speed up the process of disputing an Ontario Provincial Traffic Violation Ticket. The app uses a camera to capture an image of the ticket(s). The image is then sent to be processed and a court date is made. No hassle, paperless and easy to navigate.
